# 代码审查技能

用于执行全面代码审查的专业技能。

## 使用场景

当你需要对代码进行质量、安全性和最佳实践方面的审查时使用此技能。

## 审查清单

### 代码质量
- 代码可读性良好，组织清晰
- 函数和变量命名清晰、具有描述性
- 遵循 DRY（不要重复自己）原则

### 安全性
- 没有硬编码的密钥或凭据
- 输入验证已正确实现

### 性能
- 没有不必要的循环或计算
- 数据库查询已优化

### 测试
- 单元测试覆盖关键功能
- 边界情况已测试
