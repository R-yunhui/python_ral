# 面试题索引

> **文档风格（已统一）：** 各 `*-interview.md` 均含 **开篇定位/开胃**、**分层或表格式「答」**、**（基础补充）** 块、**场景题**、**面经补充** 与 **自测表**；与 `redis/redis-interview.md` 同一复习粒度。  
> **复习顺序：** **主干编号题**（口述结论 + 1 个追问）→ **场景题** → **面经** → **自测表** 闭卷。版本以 **官方 Release Notes** 与生产为准。

| 目录 | 内容 |
|------|------|
| [mysql](./mysql/) | MySQL（索引 + 事务/日志/MVCC + 场景 + **面经普通题补充**）；语境 **8.0 / 8.4 LTS** |
| [jvm](./jvm/) | JVM（内存、GC、类加载、JMM、诊断、场景 + **面经补充**）；语境 **JDK 8～25** |
| [java-basics](./java-basics/) | Java 基础（集合、并发、新特性、场景 + **面经补充**）；语境 **JDK 8～25** |
| [kafka](./kafka/) | Kafka（架构、ISR、生产消费、EOS、KRaft、场景 + **面经 46～58**）；**4.0 起 KRaft-only** |
| [redis](./redis/) | Redis（类型、持久化、集群、缓存、锁、场景 + **面经补充**） |
| [spring](./spring/) | Spring / Spring Boot / Spring Cloud（IoC、AOP、事务、自动配置、微服务组件、网关容错、可观测性 + **面经补充**） |
